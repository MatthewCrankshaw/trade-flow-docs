    - `@Controller("v1/public")` -- NO `@UseGuards(JwtAuthGuard)` anywhere

    **Step-by-step logic:**
    ```
    1. const tokenDto = await this.quoteTokenRetriever.findByToken(token)
    2. If null: throw HttpException with createErrorResponse([{ code: "RESOURCE_NOT_FOUND", message: "Quote not found.", details: "" }]), HttpStatus.NOT_FOUND
    3. If retriever.isExpired(tokenDto) or retriever.isRevoked(tokenDto):
       throw HttpException with createErrorResponse([{ code: "TOKEN_EXPIRED", message: "This quote link has expired. Please contact the tradesperson for an updated link.", details: "" }]), HttpStatus.GONE
    4. const quote = await this.quoteRepository.findByIdOrFail(tokenDto.quoteId)
    5. If quote.status === QuoteStatus.DELETED:
       throw HttpException with createErrorResponse([{ code: "QUOTE_UNAVAILABLE", message: "This quote is no longer available.", details: "" }]), HttpStatus.GONE
    6. Enrich: const [business, customer, job] = await Promise.all([
         this.businessRepository.findByIdOrFail(quote.businessId),
         this.customerRepository.findByIdOrFail(quote.customerId),
         this.jobRepository.findByIdOrFail(quote.jobId),
       ])
    7. Map to IPublicQuoteResponse via mapToPublicResponse(quote, business.name, customer.name, job.title)
    8. Return createResponse([response])
    ```
