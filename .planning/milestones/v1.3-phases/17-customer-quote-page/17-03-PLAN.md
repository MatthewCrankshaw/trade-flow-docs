**1. `trade-flow-api/src/quote-token/guards/quote-session-auth.guard.ts`**

Create `QuoteSessionAuthGuard` implementing `CanActivate` from `@nestjs/common`:

- Class decorated with `@Injectable()`
- Constructor injects: `QuoteTokenRetriever`, `QuoteTokenRepository`, `QuoteRepository` (from `@quote/repositories/quote.repository`), `BusinessRepository` (from `@business/repositories/business.repository`)
- `canActivate(context: ExecutionContext): Promise<boolean>` method that:
  1. Gets request via `context.switchToHttp().getRequest()`
  2. Extracts token from `request.params.token` (the `:token` route param)
  3. Calls `this.quoteTokenRetriever.findByToken(token)`
  4. If null: throws `new HttpException(createErrorResponse([{ code: "RESOURCE_NOT_FOUND", message: "Quote not found.", details: "" }]), HttpStatus.NOT_FOUND)`
  5. If `this.quoteTokenRetriever.isExpired(tokenDto)` or `this.quoteTokenRetriever.isRevoked(tokenDto)`:
     - Looks up business name: `const quote = await this.quoteRepository.findByIdOrFail(tokenDto.quoteId)`, then `const business = await this.businessRepository.findByIdOrFail(quote.businessId)`, wrapped in try/catch that falls through with empty string
     - Determines code: `this.quoteTokenRetriever.isRevoked(tokenDto) ? "TOKEN_REVOKED" : "TOKEN_EXPIRED"`
     - Determines message: revoked = "This quote is no longer available.", expired = "This quote link has expired."
     - Throws `new HttpException(createErrorResponse([{ code, message, details: businessName }]), HttpStatus.GONE)`
  6. Records first view: if `!tokenDto.firstViewedAt`, calls `await this.quoteTokenRepository.updateFirstViewedAt(tokenDto.id)`
  7. Attaches token DTO to request: `request.quoteToken = tokenDto`
  8. Returns `true`

Import path aliases: `@quote-token/services/quote-token-retriever.service`, `@quote-token/repositories/quote-token.repository`, `@quote/repositories/quote.repository`, `@business/repositories/business.repository`, `@core/response/create-response.utility`
