   - Type: `builder.mutation<Schedule, { businessId: string; jobId: string; scheduleId: string; data: { notes?: string | null } }>`
   - URL: `/v1/business/${businessId}/job/${jobId}/schedule/${scheduleId}`
   - Method: PATCH
   - transformResponse: extract `response.data[0]` (same pattern as createSchedule)
   - invalidatesTags: `[{ type: "Schedule", id: scheduleId }, { type: "Schedule", id: \`JOB-${jobId}\` }]`

   - Add `useUpdateScheduleMutation` import

   - Add `isUpdating: boolean` to return type
   - Keep existing `createSchedule` untouched

3. **hooks/index.ts** -- No change needed (already exports useScheduleActions)

4. **schedule-utils.ts** -- Create display utilities:
   - `STATUS_CONFIG: Record<ScheduleStatus, { label: string; className: string }>` -- Color-coded badge styles per user decision:
     - scheduled: blue (bg-blue-100 text-blue-800, dark mode: bg-blue-900/30 text-blue-400)
     - confirmed: green (bg-green-100 text-green-800, dark mode)
     - completed: gray (bg-gray-100 text-gray-800, dark mode)
     - canceled: red (bg-red-100 text-red-800, dark mode)
     - no-show: amber (bg-amber-100 text-amber-800, dark mode)
   - `formatScheduleDate(isoString: string): string` -- Using date-fns: "Today", "Tomorrow", or "EEE d MMM" (e.g., "Tue 10 Mar")
   - `formatScheduleTime(isoString: string): string` -- Using date-fns: "h:mm a" (e.g., "9:00 AM")
   - `formatDuration(minutes: number): string` -- "<60 min" as "{n} min", whole hours as "{n} hr(s)", fractional as "{n} hrs"
   - `isMutedStatus(status: ScheduleStatus): boolean` -- returns true for "canceled" and "no-show"
   - `groupSchedules(schedules: Schedule[]): { upcoming: Schedule[]; past: Schedule[] }` -- Split by startDateTime vs now. Upcoming: future dates ascending. Past: past dates descending.

2. **ScheduleTable.tsx** -- Desktop table view:
   - Props: `{ schedules: Schedule[]; visitTypeMap: Map<string, VisitType>; onSelect: (schedule: Schedule) => void }`
   - Columns: Date, Time, Duration, Visit Type, Status
   - Date column: `formatScheduleDate(s.startDateTime)` -- font-medium
   - Time column: `formatScheduleTime(s.startDateTime)`
   - Duration column: `formatDuration(s.durationMinutes)`
   - Visit Type column: color dot (8x8 rounded-full div with visitType.color as bg) + name. Guard null visitTypeId with fallback "General Visit" (no dot)

   - Muted styling: `className={cn(isMutedStatus(s.status) && "opacity-60")}` on TableRow
