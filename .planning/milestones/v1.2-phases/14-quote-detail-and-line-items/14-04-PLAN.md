**Step 2: Add pricing strategy selection to QuoteLineItemsCard**
Read `QuoteLineItemsCard.tsx` first to understand its current structure. It uses SearchableItemPicker's `onSelect` callback to immediately call `addLineItem`.

Modify the flow so that when a bundle item is selected (check `item.type === "bundle"`), instead of immediately adding, show a small inline UI (below the SearchableItemPicker button or as a popover/dialog) that lets the user choose:
- "Component-based pricing" (value: `"component_based"`) -- price is sum of components
- "Fixed pricing" (value: `"fixed"`) -- uses the bundle's fixed price

Use a simple approach: After a bundle is selected, show a div with two radio-style buttons (using shadcn Button with variant toggling) and a "Confirm" button. Store the selected item in local state. For non-bundle items, continue adding immediately as before.

Implementation pattern:
```tsx
const [pendingBundleItem, setPendingBundleItem] = useState<Item | null>(null);
const [selectedStrategy, setSelectedStrategy] = useState<string>("component_based");

const handleItemSelect = (item: Item) => {
  if (item.type === "bundle") {
    setPendingBundleItem(item);
    setSelectedStrategy(item.bundleConfig?.priceStrategy ?? "component_based");
  } else {
    addLineItem({ businessId, quoteId, itemId: item.id, quantity: 1 });
  }
};

const handleConfirmBundle = () => {
  if (!pendingBundleItem) return;
  addLineItem({
    businessId, quoteId,
    itemId: pendingBundleItem.id,
    quantity: 1,
    priceStrategy: selectedStrategy,
  });
  setPendingBundleItem(null);
};

const handleCancelBundle = () => {
  setPendingBundleItem(null);
};
```

For the UI, render below the SearchableItemPicker when `pendingBundleItem` is set:
```tsx
{pendingBundleItem && (
  <div className="rounded-lg border bg-muted/30 p-3 space-y-2">
    <div className="text-sm font-medium">
      Pricing for "{pendingBundleItem.name}"
    </div>
    <div className="flex gap-2">
      <Button
        type="button"
        size="sm"
        variant={selectedStrategy === "component_based" ? "default" : "outline"}
        onClick={() => setSelectedStrategy("component_based")}
      >
        Component-based
      </Button>
      <Button
        type="button"
        size="sm"
        variant={selectedStrategy === "fixed" ? "default" : "outline"}
        onClick={() => setSelectedStrategy("fixed")}
      >
        Fixed price
      </Button>
    </div>
    <div className="flex gap-2">
      <Button type="button" size="sm" onClick={handleConfirmBundle}>
        Add to quote
      </Button>
      <Button type="button" size="sm" variant="ghost" onClick={handleCancelBundle}>
        Cancel
      </Button>
    </div>
  </div>
)}
```

Import `useState` if not already imported. Import `Button` from shadcn if not already imported. Import the `Item` type if not already available.
