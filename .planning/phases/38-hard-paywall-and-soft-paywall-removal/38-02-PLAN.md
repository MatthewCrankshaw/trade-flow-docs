**For EACH feature page listed in files, apply this transformation:**

The soft paywall pattern in each feature page looks like this:
```typescript
// BEFORE:
const { openPaywall, isActive } = useSubscription();

const handleCreate = () => {
  if (!isActive) {
    openPaywall("Create a Customer");  // or similar label
    return;
  }
  // ... actual create logic
};
```

Transform to:
```typescript
// AFTER:
const handleCreate = () => {
  // ... actual create logic (no subscription check -- PaywallGuard handles access)
};
```

**Specific changes per file:**

1. Remove `useSubscription` hook calls that destructure `openPaywall` or `isActive` for gating purposes. If the hook is ONLY used for `openPaywall`/`isActive` gating, remove the entire `useSubscription()` call and its import.
2. Remove the `if (!isActive) { openPaywall(...); return; }` guard blocks from ALL write action handlers (create, edit, delete, send).
3. Remove the `import { useSubscription } from "@/features/subscription/hooks/useSubscription"` line if `useSubscription` is no longer used in the file after removing openPaywall calls.
4. Remove any `import { useAppDispatch } from "@/store/hooks"` lines that were only used for paywall dispatching (check if dispatch is used elsewhere in the file first).
