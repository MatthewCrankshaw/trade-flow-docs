## Interaction States

### PaywallModal

| State | Trigger | Visual |
|-------|---------|--------|
| Closed | Default | Not rendered |
| Open | User clicks a locked write action | Dialog overlay + modal with feature-aware title |
| Loading | "Start free trial" clicked, checkout session being created | Button shows `Loader2` spinner, disabled state |
| Redirect | Checkout URL received | `window.location.href` redirect to Stripe (page unloads) |
| Dismissed | "Maybe later" clicked | Modal closes, returns to read-only page |

### /subscribe/success Page

| State | Condition | Visual |
|-------|-----------|--------|
| Polling | `isPolling === true` | `Loader2` spinner (animate-spin), "Activating your subscription..." text |
| Verified | `isVerified === true` | `CheckCircle` icon (text-primary, h-16 w-16), "You're all set!" heading, "Go to Dashboard" button |
| Timed out | `isTimedOut === true` | Same as Verified with additional note: "Your subscription is being activated -- you'll receive a confirmation email shortly." |

### Persistent CTA

| State | Condition | Visual |
|-------|-----------|--------|
| Hidden | `isLoading === true` OR `isActive === true` | Not rendered |
| Visible (desktop) | `isLoading === false && !isActive` | Button in header area: "Unlock Full Access" |
| Visible (mobile) | `isLoading === false && !isActive` | Banner at top of page content |

### Dashboard Banner

| State | Condition | Visual |
|-------|-----------|--------|
| Hidden | `isLoading === true` OR `isActive === true` | Not rendered |
| Visible | `isLoading === false && !isActive` AND on dashboard page | `bg-muted` banner with read-only mode copy and CTA |

## Accessibility Notes

- PaywallModal uses shadcn Dialog (Radix UI): focus trap, Escape to close, aria-labelledby from DialogTitle
- All CTA buttons have descriptive text (no icon-only buttons without aria-label)
- Success page polling state has text description alongside spinner (not spinner-only)
- Persistent CTA hidden during loading prevents assistive technology from announcing stale state
- Cancel page "Try again" is a visible link/button, not just a back navigation gesture
