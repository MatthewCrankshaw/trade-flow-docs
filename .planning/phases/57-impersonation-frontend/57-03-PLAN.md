**1. Create `trade-flow-ui/src/features/support/components/ImpersonateUserDialog.tsx`**

Follow the `MarkAsLostDialog.tsx` dialog wrapper + body split pattern exactly.

**Props interface:**
```typescript
interface ImpersonateUserDialogProps {
  open: boolean;
  onOpenChange: (open: boolean) => void;
  targetUser: {
    id: string;
    name: string;
    email: string;
  };
}
```

**Dialog wrapper component:**
- Uses `Dialog` from `@/components/ui/dialog` (NOT AlertDialog -- follow existing codebase dialog pattern from MarkAsLostDialog)
- `DialogContent` with `className="sm:max-w-md"`
- Renders `{open && <ImpersonateUserDialogBody ... />}` inside content (lazy mount pattern from analog)
- Exports `ImpersonateUserDialog`

**Dialog body component (internal, not exported):**

Props: `targetUser`, `onOpenChange`

State:
- `reasonText` -- `useState("")`
- `showValidation` -- `useState(false)` (tracks whether to show validation error)

Uses `useImpersonation()` hook to get `startSession` and `isStarting`.

**handleSubmit:**
1. Trim the `reasonText`. If empty, set `showValidation(true)` and return early
2. Call `await startSession(targetUser.id, targetUser.name, targetUser.email, reasonText.trim())`
3. On success: `onOpenChange(false)` (dialog closes; navigation handled by hook)
4. On error: dialog stays open (error toast handled by hook)

**JSX structure:**
```
DialogHeader
  DialogTitle: "Impersonate User"
  DialogDescription: "You will see exactly what {targetUser.name} sees, including their data, subscription, and permissions. All actions are logged."

div.space-y-4
  div.space-y-2
    Label htmlFor="impersonation-reason": "Reason"
    Textarea
      id="impersonation-reason"
      placeholder="Why are you impersonating this user?"
      value={reasonText}
      onChange={(e) => { setReasonText(e.target.value); if (showValidation) setShowValidation(false); }}
      rows={3}
    {showValidation && reasonText.trim() === "" && (
      <p className="text-sm text-destructive">A reason is required before impersonating a user.</p>
    )}

DialogFooter
  Button variant="outline" onClick={() => onOpenChange(false)} disabled={isStarting}:
    "Keep Reviewing"
  Button variant="default" onClick={handleSubmit} disabled={isStarting}:
    {isStarting ? <><Loader2 className="mr-1 h-4 w-4 animate-spin" />Starting...</> : "Start Impersonation"}
```

- `useState` from `"react"`
- `Loader2` from `"lucide-react"`
