**2. Create `src/impersonation/repositories/impersonation-audit.repository.ts`** (per D-07, IAUD-02, IAUD-03):
- `@Injectable()` class `ImpersonationAuditRepository`
- `static readonly COLLECTION = "impersonation_audit"`
- `private readonly logger = new AppLogger(ImpersonationAuditRepository.name)`
- Constructor injects `MongoDbFetcher` and `MongoDbWriter`
- **`create(dto: IImpersonationAuditDto): Promise<IImpersonationAuditDto>`** -- builds entity with `new ObjectId(dto.id)`, `...createAuditFields()`, maps DTO fields to entity fields (DateTime to `Date` via `.toJSDate()`, null endedAt stays null). Calls `this.writer.insertOne<IImpersonationAuditEntity>(COLLECTION, entity)`. Returns `this.toDto(entity)`.
- **`findBySessionId(sessionId: string): Promise<IImpersonationAuditDto | null>`** -- calls `this.fetcher.findOne<IImpersonationAuditEntity>(COLLECTION, { _id: new ObjectId(sessionId) })`. Returns `this.toDto(entity)` or null.
- **`findBySupportUserId(supportUserId: string): Promise<IImpersonationAuditDto[]>`** -- calls fetcher to find all entries for a support user. Returns mapped DTOs.
- **`markEnded(sessionId: string, endedAt: DateTime): Promise<void>`** -- calls `this.writer.updateOne<IImpersonationAuditEntity>(COLLECTION, { _id: new ObjectId(sessionId) } as never, { $set: { endedAt: endedAt.toJSDate(), status: ImpersonationStatus.ENDED, updatedAt: new Date() } } as never)`. This is the ONLY mutation method -- a controlled single-field update for session termination.
- **`private toDto(entity: IImpersonationAuditEntity): IImpersonationAuditDto`** -- converts `_id` to string `id`, Date fields to `DateTime.fromJSDate()`, null endedAt stays null, status cast to `ImpersonationStatus`.
- **NO general `update()`, `delete()`, `updateOne()`, or `deleteOne()` methods.** The class intentionally omits these to enforce append-only semantics at the application layer.
