<objective>
Add tax rate ID validation to item creation and update flows, ensuring the referenced tax rate exists, belongs to the same business, and is active (enabled).

Purpose: Items must not reference invalid or unauthorized tax rates. This enforces data integrity at the service layer, following the established schedule-to-visit-type validation pattern.

Output: ItemCreatorService and ItemUpdaterService validate taxRateId via TaxRateRetrieverService. All validation scenarios tested.
</objective>

       ```typescript
       private async validateTaxRateIfChanged(authUser: IUserDto, existing: IItemDto, updates: IItemDto): Promise<void> {
         // Skip validation if taxRateId hasn't changed
         if (updates.taxRateId === existing.taxRateId) {
           return;
         }

         // For bundle items, taxRateId must remain null
         if (existing.type === ItemType.BUNDLE) {
           if (updates.taxRateId !== null) {
             throw new InvalidRequestError(
               ErrorCodes.DEFAULT_TAX_RATE_NOT_ALLOWED,
               `Tax rate ID is not allowed for bundle items`,
             );
           }
           return;
         }

         // Non-bundle: new taxRateId must be valid
         if (updates.taxRateId === null) {
           throw new InvalidRequestError(
             ErrorCodes.INVALID_ITEM_DEFAULT_TAX_RATE,
             `Tax rate ID is required for ${existing.type} items`,
           );
         }

         const taxRate = await this.taxRateRetriever.findByIdOrFail(authUser, updates.taxRateId);

         if (taxRate.status !== TaxRateStatus.ENABLED) {
           throw new InvalidRequestError(
             ErrorCodes.INVALID_ITEM_DEFAULT_TAX_RATE,
             `Tax rate must be active to assign to an item`,
           );
         }
       }
       ```

    3. Call `validateTaxRateIfChanged` in the `update` method BEFORE the repository update:

       public async update(authUser: IUserDto, existing: IItemDto, updates: IItemDto): Promise<IItemDto> {
         const accessController = this.accessControllerFactory.create(this.itemPolicy);
         accessController.canUpdate(authUser, existing);

         await this.validateTaxRateIfChanged(authUser, existing, updates);

         const updatedItem = await this.itemRepository.update(updates);
         return updatedItem;
       }
