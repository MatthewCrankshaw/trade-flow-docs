**Add RTK Query endpoints to `src/features/support/api/supportApi.ts`:**

Add `getSupportUserDetail` query:
```typescript
getSupportUserDetail: builder.query<SupportUserDetail, string>({
  query: (id) => `/v1/support/users/${id}`,
  transformResponse: (response: StandardResponse<SupportUserDetail>) => response.data[0],
  providesTags: (_result, _error, id) => [{ type: "SupportUser", id }],
}),
```

Add `getDashboardMetrics` query:
```typescript
getDashboardMetrics: builder.query<DashboardMetrics, void>({
  query: () => `/v1/support/dashboard/metrics`,
  transformResponse: (response: StandardResponse<DashboardMetrics>) => response.data[0],
  providesTags: [{ type: "DashboardMetrics", id: "METRICS" }],
}),
```

Export: `useGetSupportUserDetailQuery`, `useGetDashboardMetricsQuery`.
