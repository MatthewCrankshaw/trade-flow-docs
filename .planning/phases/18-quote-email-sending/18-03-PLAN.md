**1. Create QuoteEmailSettings** (`src/features/business/components/QuoteEmailSettings.tsx`):
Per UI-SPEC Business Settings -- Quote Email Template Contract:

Props interface:
```typescript
interface QuoteEmailSettingsProps {
  businessId: string;
  defaultSubject: string;  // Current value from business or system default
  defaultMessage: string;  // Current value from business or system default
}
```

Internal state:
- `subject` state: initialized from `defaultSubject`
- `message` state: initialized from `defaultMessage`
- Use `useUpdateBusinessMutation` from businessApi

System defaults (for placeholder text):
- Subject placeholder: `"Quote {{quoteNumber}} for {{jobTitle}}"`
- Message placeholder: `"Hi {{customerName}},\n\nPlease find your quote {{quoteNumber}} for {{jobTitle}}.\n\nKind regards,\n{{userName}}"`

Layout (per UI-SPEC wireframe):
```tsx
<Card>
  <CardHeader>
    <CardTitle>Quote Email Template</CardTitle>
    <CardDescription>Configure the default email sent with your quotes.</CardDescription>
  </CardHeader>
  <CardContent className="space-y-4">
    {/* Default Subject */}
    <div className="space-y-2">
      <Label htmlFor="quoteEmailSubject">Default Subject</Label>
      <Input
        id="quoteEmailSubject"
        value={subject}
        onChange={(e) => setSubject(e.target.value)}
        placeholder="Quote {{quoteNumber}} for {{jobTitle}}"
      />
    </div>

    {/* Default Message */}
    <div className="space-y-2">
      <Label htmlFor="quoteEmailBody">Default Message</Label>
      <Textarea
        id="quoteEmailBody"
        value={message}
        onChange={(e) => setMessage(e.target.value)}
        placeholder="Hi {{customerName}},&#10;&#10;Please find your quote {{quoteNumber}} for {{jobTitle}}.&#10;&#10;Kind regards,&#10;{{userName}}"
        rows={8}
      />
    </div>

    {/* Available variables as Badge elements */}
    <div className="space-y-2">
      <p className="text-xs text-muted-foreground">Available variables:</p>
      <div className="flex flex-wrap gap-2">
        <Badge variant="outline" className="text-xs">{"{{customerName}}"}</Badge>
        <Badge variant="outline" className="text-xs">{"{{quoteNumber}}"}</Badge>
        <Badge variant="outline" className="text-xs">{"{{jobTitle}}"}</Badge>
        <Badge variant="outline" className="text-xs">{"{{businessName}}"}</Badge>
        <Badge variant="outline" className="text-xs">{"{{userName}}"}</Badge>
      </div>
      <p className="text-xs text-muted-foreground">
        Variables are replaced with actual values when the send dialog opens.
      </p>
    </div>

    <Separator className="my-4" />

    {/* Save button */}
    <div className="flex justify-end">
      <Button onClick={handleSave} disabled={isSaving}>
        {isSaving ? "Saving..." : "Save Template"}
      </Button>
    </div>
  </CardContent>
</Card>
```

handleSave function:
```typescript
const handleSave = async () => {
  try {
    await updateBusiness({
      id: businessId,
      data: {
        quoteEmailSubject: subject || undefined,
        quoteEmailBody: message || undefined,
      },
    }).unwrap();
    toast.success("Quote email template saved");
  } catch {
    toast.error("Failed to save template. Please try again.");
  }
};
```

Import toast from `@/lib/toast`.
